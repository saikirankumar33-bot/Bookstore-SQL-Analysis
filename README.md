# 📚 Bookstore Sales & Inventory Analysis (SQL Project)

## 📌 Objective
Analyze bookstore data to understand sales trends, customer behavior, and inventory performance.

## 🗂️ Dataset
The project uses 3 tables:
- Books
- Customers
- Orders

## 🏗️ Database Schema
Includes tables for books, customers, and orders with relationships.

## 🔍 Key Analysis
- Top selling books & genres
- Customer order patterns
- Revenue analysis
- Stock remaining after sales

## 📊 Key Insights
- Repeat customers drive revenue
- Some books need restocking
- Certain genres perform better

## 🛠️ Tools Used
- SQL (PostgreSQL)
- Excel

## 🚀 How to Run
1. Create tables using schema.sql  
2. Import CSV data  
3. Run queries.sql
